## 선택 요소 — 신뢰 배지 (default.png 한정)

홈 OG 이미지에만 우상단 또는 제목 옆에 작은 배지 추가 가능:

```
┌──────────────────────┐
│  ✓ 식약처 기준 적용  │   배경: #064E3B (다크 그린)
│  ✓ 16항목 자동 검토  │   텍스트: #A7F3D0 (민트 그린)
└──────────────────────┘
```

- 크기: 텍스트 14px, 패딩 8px × 16px, radius 20px (pill 형태)
- 배지는 2개 이하로 제한
