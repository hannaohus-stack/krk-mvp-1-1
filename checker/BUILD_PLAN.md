### Day 15 · 전체 UX 점검 + 모바일 최적화

**Claude Code 프롬프트:**
```

회원가입 → Dashboard → Creator → Review → 결제 → Export

체크:
- 페이지 전환 애니메이션 자연스러운지

1. 일반: 딸기잼 원재료 3개 → 면제 적용 → 전부 🟢 → 결제 → PDF 3종 (목표 3분 이내)
2. 자동 감지: 땅콩버터 입력 → 알레르기 ✓ + 복합원재료 ✓ → 세부 입력 → 라벨 반영
3. 위반: 원재료 순서 오류 → 🔴 감지 → [수정하기] → 정렬 → 🟢 → 다운로드
```
