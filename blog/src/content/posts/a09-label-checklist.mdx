라벨 디자인을 의뢰할 때 정보가 빠지면 시안 수정 라운드가 두세 번씩 늘어납니다. 결국 인쇄 일정이 밀리고 출시 시점이 흔들리는 경우가 많습니다.

이 글은 1인 식품 제조사가 디자이너에게 시안을 넘기기 전, **한 번에 확인해야 할 모든 정보**를 카테고리별로 정리한 체크리스트입니다. 항목 옆 박스를 따라가며 빠진 부분이 없는지 점검해 보세요.

---

## 기본 표시사항 체크리스트

아래 항목은 모든 가공식품에 공통으로 필요한 의무 표시사항입니다. ([9가지 의무 표시사항 자세히 →](/a01-label-9-items))

### 제품 정보

- [ ] **제품명** — 브랜드명과 구분된 법적 제품명 (예: `차차 그래놀라`가 아닌 `오트 그래놀라`)
- [ ] **식품유형** — 품목제조보고서 기재 유형과 일치
- [ ] **내용량** — 단위 포함 (g, ml, 개 등)

### 제조·판매원 정보

- [ ] 제조원 명칭 및 소재지
- [ ] 판매원 명칭 및 소재지 (제조원과 다를 경우)
- [ ] 위탁자 명칭 (위탁 제조인 경우)

### 날짜 표시

- [ ] 유통기한 / 품질유지기한 / 제조연월일 중 해당 항목 결정
- [ ] 표기 형식 확인 (YYYY.MM.DD 또는 별도 지정 위치)

### 성분 정보

- [ ] **원재료명** — 함량 많은 순으로 정렬 완료 ([함량순 표기 자세히 →](/a02-ingredient-order))
- [ ] **알레르기 유발물질** — 22종 포함 여부 확인 ([22품목 자세히 →](/a03-allergen-22))
- [ ] 알레르기 표시 방법 결정 (굵은 글씨 강조 또는 별도 표기)

### 보관·취급

- [ ] 보관 방법 문구 확정
- [ ] 개봉 후 취급 방법 (해당 시)

### 영양성분 (해당 시)

- [ ] 영양표시 의무 대상 여부 확인
- [ ] **강조 표시 사용 여부 확인** — `무가당`, `저지방` 등 사용 시 영양성분표 필수 ([강조 표시 자세히 →](/a04-nutrition-emphasis))
- [ ] 영양성분 분석 완료 여부

## 분리배출·재질 표시 체크리스트

소재가 두 가지 이상이면 마크도 두 개 이상 필요합니다. ([분리배출 마크 자세히 →](/a05-recycle-mark))

- [ ] 포장재 소재 목록 확정 (본체 + 뚜껑 + 부속재)
- [ ] 각 소재별 분리배출 마크 파일 준비
- [ ] 용기·포장 재질 표기 문구 작성

> **놓치기 쉬운 부분** — 유리병에 플라스틱 캡을 결합하는 경우, 두 가지 모두 표기해야 합니다. ([유리병+캡 표시법 →](/a06-mixed-packaging))

## 법정 주의 문구 체크리스트

제품 유형에 따라 추가 문구가 필요할 수 있습니다.

- [ ] 유아식, 특수의료용도식품 등 해당 시 별도 주의 문구
- [ ] 알코올 포함 식품: 알코올 함량 표기
- [ ] 카페인 고함유 식품: 카페인 함량 및 주의 문구
- [ ] 면류 등: 조리 방법 (해당 시)

## 원산지 표시 체크리스트

원재료 원산지 표시 의무는 별도 법령(원산지표시법)이 적용됩니다. ([원산지 표시 자세히 →](/a07-origin-label))

- [ ] 농수산물 원재료 원산지 확인
- [ ] 수입 원재료 사용 비율 점검
- [ ] 원산지 표기 형식 결정

## 디자이너에게 전달할 파일 목록

체크리스트 완성 후 아래 자료를 한꺼번에 전달하면 시안 작업이 매끄럽습니다.

- [ ] 위 체크리스트 완성본 (텍스트 파일 또는 문서)
- [ ] 분리배출 마크 파일 (소재별)
- [ ] 제조원·판매원 서명 또는 인감 (필요 시)
- [ ] 영양성분표 수치 (분석 결과)
- [ ] 브랜드 가이드 또는 색상 코드
- [ ] 참고 라벨 (기존 자사 제품 또는 레퍼런스)

## 시안 받은 후 한 번 더 점검

디자이너가 시안을 보내주면, 인쇄 발주 전에 한 번 더 자율 점검하는 것을 권장합니다.

> 📍 **시안 자동 점검하기** → [checker.krk.team](https://checker.krk.team)
> 인쇄 발주 전 1~2분 점검으로 폐기 비용을 막을 수 있습니다.

KRK Checker로 점검하면 신호등 결과로 누락·오류 항목을 빠르게 확인할 수 있습니다. ([결과 화면 보는 법 →](/a10-krk-result))

## 자주 묻는 질문

**Q. 체크리스트를 디자이너에게 그대로 전달해도 되나요?**
네. 정보가 카테고리별로 정리되어 있어 디자이너가 시안 작업하기 수월합니다. 빠진 부분을 디자이너가 다시 묻지 않아도 됩니다.

**Q. 즉석판매제조가공업도 이 체크리스트가 모두 필요한가요?**
업종에 따라 의무 표시 항목이 다릅니다. 즉석판매제조가공업은 간소화된 의무가 적용됩니다. ([업종별 차이 자세히 →](/a08-business-types))

**Q. 시안 받은 후 점검은 꼭 필요한가요?**
디자인 과정에서 텍스트 오기, 위치 이동, 단위 누락 등이 자주 발생합니다. 인쇄 후 폐기 비용을 막기 위해 발주 전 한 번 더 확인하는 것을 강력 권장합니다.
