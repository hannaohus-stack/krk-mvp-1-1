디자이너에게 라벨 시안을 요청하기 전에, 표시 정보가 완전한지 확인하는 과정이 필요합니다. 정보가 빠진 채로 시안이 나오면 수정 라운드가 늘어납니다.

## 기본 표시사항 체크리스트

아래 항목은 모든 가공식품에 공통으로 필요합니다.

### 제품 정보
- [ ] 제품명 (브랜드명과 구분된 법적 제품명)
- [ ] 식품유형 (품목제조보고서 기재 유형과 일치)
- [ ] 내용량 (단위 포함: g, ml, 개 등)

### 제조·판매원 정보
- [ ] 제조원 명칭 및 소재지
- [ ] 판매원 명칭 및 소재지 (제조원과 다를 경우)
- [ ] 위탁자 명칭 (위탁 제조인 경우)

### 날짜 표시
- [ ] 유통기한 / 품질유지기한 / 제조연월일 중 해당 항목 결정
- [ ] 표기 형식 확인 (YYYY.MM.DD 또는 별도 지정 위치)

### 성분 정보
- [ ] 원재료명 (함량 많은 순 정렬 완료)
- [ ] 알레르기 유발물질 (22종 포함 여부 확인)
- [ ] 알레르기 표시 방법 결정 (굵은 글씨 강조 또는 별도 표기)

### 보관·취급
- [ ] 보관 방법 문구 확정
- [ ] 개봉 후 취급 방법 (해당 시)

### 영양성분 (해당 시)
- [ ] 영양표시 의무 대상 여부 확인
- [ ] 강조 표시 사용 여부 확인
- [ ] 영양성분 분석 완료 여부

## 분리배출·재질 표시 체크리스트

- [ ] 포장재 소재 목록 확정 (본체 + 뚜껑 + 부속재)
- [ ] 각 소재별 분리배출 마크 파일 준비
- [ ] 용기·포장 재질 표기 문구 작성

## 법정 주의 문구 체크리스트

제품 유형에 따라 추가 문구가 필요할 수 있습니다.

- [ ] 유아식, 특수의료용도식품 등 해당 시 별도 주의 문구
- [ ] 알코올 포함 식품: 알코올 함량 표기
- [ ] 카페인 고함유 식품: 카페인 함량 및 주의 문구
- [ ] 면류 등: 조리 방법 (해당 시)

## 디자이너에게 전달할 파일 목록

- [ ] 위 체크리스트 완성본 (텍스트 파일 또는 문서)
- [ ] 분리배출 마크 파일 (소재별)
- [ ] 제조원·판매원 서명 또는 인감 (필요 시)
- [ ] 영양성분표 수치 (분석 결과)
- [ ] 브랜드 가이드 또는 색상 코드

---

체크리스트 모든 항목을 채운 뒤 디자이너에게 넘기면, 시안 수정 횟수가 줄고 납기도 지킬 수 있습니다. KRK Checker의 자율 점검 결과를 이 체크리스트 작성에 참고할 수 있습니다.

> ⚠️ 이 글은 자율 점검을 돕기 위한 참고용입니다. 개별 제품에 대한 법률 자문이 아닙니다.
