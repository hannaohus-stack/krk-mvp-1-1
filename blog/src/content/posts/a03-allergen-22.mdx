식품 알레르기 표시는 소비자 안전과 직결되는 의무 표시사항입니다. 식약처가 지정한 **22품목은 법정 의무 표시 대상**이며, 누락 시 회수 명령을 받을 수 있습니다.

이 글은 22품목 전체 목록과 1인 식품 제조사가 라벨에서 자주 놓치는 부분 — 가공보조제, 유사 원재료, 교차오염 우려 — 을 정리합니다.

---

## 알레르기 유발물질 22품목 전체 목록

식약처 고시 2024-66호 기준 22품목은 다음과 같습니다.

| 분류 | 품목 |
|---|---|
| 알류·가금류 | 알류(가금류), 닭고기 |
| 유제품 | 우유 |
| 곡류·콩 | 메밀, 대두, 밀 |
| 견과류·씨앗 | 땅콩, 호두, 잣, 아몬드, 캐슈넛, 피스타치오 |
| 어패류 | 고등어, 게, 새우, 오징어, 조개류(굴·전복·홍합 포함) |
| 육류 | 돼지고기, 쇠고기 |
| 과일 | 복숭아, 토마토 |
| 화학물질 | 아황산류(SO₂, 10mg/kg 이상) |

**총 22품목**입니다.

> **놓치기 쉬운 부분** — 아황산류는 와인, 건과일, 일부 가공식품에 보존제로 사용되는데, 라벨 표시 의무 인지가 낮아 누락이 잦습니다.

---

## 놓치기 쉬운 3가지 케이스

### 1. 가공보조제

제조 과정에서 사용되었지만 최종 제품에 잔류할 수 있는 가공보조제도 알레르기 표시 대상입니다.

예) 유화제로 대두 레시틴 사용 → **`대두` 표시 의무**
예) 효소제로 우유 단백질 유래 사용 → **`우유` 표시 의무**

> **놓치기 쉬운 부분** — 가공보조제는 원재료명 표시란에서 생략 가능한 경우가 있지만, **알레르기 표시는 별도 의무**입니다.

### 2. 유사 원재료

특정 부위나 형태가 달라도 같은 알레르기원으로 봅니다.

- 달걀흰자만 사용 → `알류` 표시
- 우유에서 분리한 유청 → `우유` 표시
- 새우젓 → `새우` 표시
- 새우 추출액 → `새우` 표시

부분이나 가공 형태가 달라도 원료가 22품목에 해당하면 표시 대상입니다.

### 3. 교차오염 우려

동일 설비에서 다른 알레르기 원재료를 취급하는 경우, **주의 표시를 권고**합니다.

예) `이 제품은 견과류를 함께 사용하는 시설에서 제조되었습니다.`

법정 의무는 아니지만 소비자 안전 측면에서 표기를 권장합니다.

## 표기 방법

원재료명 표시란 **하단**에 **굵은 글씨** 또는 **별도 칸**으로 알레르기 유발물질을 표시합니다.

표기 예시:

```
원재료명: 밀가루(밀), 설탕, 우유, 계란노른자(알류), 버터(우유), 식물성유지(대두유)...

알레르기 유발물질: 밀, 우유, 알류, 대두 함유
```

**별도 표기란을 만들고 22품목 중 사용된 항목만 정확히 명시**하는 것이 가장 명확한 표기입니다.

## 자주 묻는 질문

**Q. 5% 미만으로 사용해도 표시해야 하나요?**
네. 함량에 관계없이 22품목에 해당하면 표시 의무가 있습니다. 5% 미만 원재료 예외 규정은 알레르기 표시에는 적용되지 않습니다.

**Q. "함유 가능성" 표시는 어떻게 하나요?**
법정 의무는 아니지만, 교차오염 우려가 있는 경우 별도 문구로 안내하는 것을 권장합니다. 예) `이 제품은 견과류를 사용하는 시설에서 제조되었습니다.`

**Q. 수입 가공식품도 22품목 기준이 동일한가요?**
국내 판매되는 모든 가공식품은 식약처 고시 22품목 기준을 따릅니다. 수입 원산지 기준이 더 적은 경우에도 국내 기준에 맞춰 추가 표시해야 합니다.

**Q. 22품목에 해당하지 않는 알레르기는 표시하지 않아도 되나요?**
법정 의무는 22품목만 적용됩니다. 다만 소비자 안전을 위해 자율적으로 표시하는 것을 권장합니다.

## 22품목 자율 점검 체크리스트

라벨 확정 전 아래 항목을 확인해 보세요.

- [ ] 사용한 원재료 중 22품목 해당 여부 확인
- [ ] 가공보조제에 22품목 포함 여부 확인
- [ ] 유사 원재료(부위·가공형태 다른 경우) 확인
- [ ] 알레르기 별도 표기란 작성 완료
- [ ] 교차오염 우려 시 주의 문구 추가 검토
