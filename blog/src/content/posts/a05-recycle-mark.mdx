분리배출 마크는 "그냥 달아두면 되는 것"처럼 보이지만, 소재별 개별 표시 원칙을 모르면 시정 명령 대상이 됩니다. 특히 복합 포장재를 쓰는 1인 식품 제조사에서 혼동이 가장 많은 항목 중 하나입니다.

이 글은 환경부 고시 기준 분리배출 의무 대상, 소재별 표시 원칙, 복합 포장재 처리 방법, 표시 위치와 크기까지 정리합니다.

---

## 분리배출 표시 의무 대상

환경부 고시에 따른 의무 대상은 다음과 같습니다.

- **합성수지(플라스틱)** — PET, PP, PE, PS, PVC 등
- **금속 캔** — 알루미늄 캔, 철 캔
- **유리병**
- **종이팩** — 일반 종이팩, 멸균팩 포함
- **발포합성수지** — 스티로폼

> **놓치기 쉬운 부분** — 비닐류(LDPE 봉투, 지퍼백 등)도 합성수지로 분류되어 표시 의무가 있습니다. "비닐이라서 플라스틱이 아니다"는 착각이 많은 부분입니다.

---

## 핵심 원칙 — 소재가 다르면 마크도 따로

분리배출 마크의 가장 중요한 원칙입니다. **소재가 두 가지 이상이면 마크도 두 개 이상** 달아야 합니다. 혼합 표기나 한쪽 생략은 불가합니다.

예) 유리병 + 금속 뚜껑 조합:
- 유리병 → **유리** 분리배출 마크
- 금속 뚜껑 → **캔·고철** 분리배출 마크

> 📍 **내 라벨 분리배출 마크 자동 점검** → [checker.krk.team](https://checker.krk.team)

---

## 포장재 조합별 표시 예시

| 포장재 조합 | 표시 방법 |
|---|---|
| 유리병 + 플라스틱 캡 | 유리 + 플라스틱 (별도 표시) |
| 종이박스 + 비닐 창 | 종이 + 비닐류 (별도 표시) |
| 알루미늄 파우치 | 비닐류 (단독 표시) |
| 크라프트 지퍼백 | 종이 + 비닐류 (재질별 분리 표시) |
| PET 병 + PP 캡 | 플라스틱 마크 2개 (재질 다르면 각각) |

> **놓치기 쉬운 부분** — 크라프트 지퍼백처럼 한 포장재 안에 소재가 혼합된 경우, 주 소재(종이)와 부속 소재(지퍼 비닐)를 구분해서 표시해야 합니다. "비닐류" 하나만 달면 부족합니다.

유리병 + 플라스틱 캡 조합의 자세한 표시 방법은 별도 글에 정리했습니다. ([유리병+캡 표시법 →](/a06-mixed-packaging))

## 표시 위치와 크기

- **표시 위치:** 포장재 주 표시면 또는 정보표시면
- **최소 크기:** 8mm × 8mm 이상 (소형 포장재는 예외 규정 별도 확인)
- **소재명 텍스트:** 마크 옆에 반드시 함께 기재 (`플라스틱`, `유리`, `캔·고철` 등)

> **놓치기 쉬운 부분** — 마크 이미지만 달고 소재명 텍스트를 빠뜨리는 경우가 많습니다. 마크 + 텍스트가 세트입니다.

---

## 공식 마크 파일 구하는 방법

인쇄소에 라벨을 발주할 때 분리배출 마크 파일을 별도로 전달해야 합니다.

공식 마크 파일 무료 다운로드:
- [한국포장재재활용사업공제조합](https://www.kora.or.kr)
- [환경부 자원순환정보시스템](https://www.recycling-info.or.kr)

> **실무 팁** — 임의로 제작한 마크 디자인은 사용할 수 없습니다. 반드시 공식 파일을 사용해야 합니다.

---

## 포장재 소재 변경 시 반드시 업데이트

포장재 소재가 바뀌면(예: 유리 → PET 병) 분리배출 마크도 함께 업데이트해야 합니다. 라벨 리디자인 때 빠뜨리기 가장 쉬운 항목 중 하나입니다.

변경 시 체크리스트:
- [ ] 포장재 소재 목록 업데이트
- [ ] 분리배출 마크 파일 교체
- [ ] 재질 표기 텍스트 수정
- [ ] 라벨 파일 최종 검수

---

## 분리배출 마크 자율 점검 체크리스트

- [ ] 의무 대상 포장재 소재 목록 확인
- [ ] 소재별 개별 마크 준비 (혼합 표기 금지)
- [ ] 공식 마크 파일 사용 여부 확인
- [ ] 마크 최소 크기 8mm × 8mm 이상
- [ ] 마크 옆 소재명 텍스트 포함
- [ ] 정보표시면 배치 완료

체크리스트 전체 버전은 별도 글에 정리했습니다. ([디자이너 전달 체크리스트 →](/a09-label-checklist))

## 자주 묻는 질문

**Q. 소형 포장재는 마크 크기 예외가 있나요?**
네. 포장재 면적이 일정 기준 이하인 경우 크기 예외 규정이 있습니다. 정확한 기준은 환경부 고시를 확인하거나 한국포장재재활용사업공제조합에 문의하세요.

**Q. 재활용 불가 포장재는 어떻게 표시하나요?**
재활용이 어려운 경우 "재활용 어려움" 표시를 해야 하는 품목이 별도로 있습니다. 의무 대상 여부는 환경부 고시를 확인하세요.

**Q. 비닐 포장재에 분리배출 마크가 필요한가요?**
LDPE 봉투, 지퍼백, 비닐 파우치 등은 합성수지(비닐류)로 분류되어 표시 의무가 있습니다.
