# KRK Checker Blog — Frontend Design Brief

> **From**: Hanna
> **To**: Claude Design
> **Date**: 2026-05-24
> **Scope**: 리스트 + 상세 페이지 / Desktop + Mobile

---

## 1. 한 줄 정의

KRK Checker 블로그는 **스몰 식품 브랜드가 판매 전 라벨 기준을 이해하고, KRK Checker로 자연스럽게 전환하도록 돕는 지식 허브**다.

---

## 2. 목적 / 포지셔닝

- **Phase 1 런칭 보조 자산** (제품 기능 X, 브랜드 퍼널 O)
- 역할: 신뢰 형성 · 검색 유입 · 구매 전 교육
- 포지션: ❌ 식품행정 정보 블로그 → ✅ 스몰 식품 브랜드용 판매 전 라벨 기준 가이드
- 전환 목적: 블로그 글 → **KRK Checker 시작하기**
- 톤: 행정 설명서 ❌ → 정돈된 editorial 가이드 ✅

**슬로건 (사용 가능)**
> 라벨도, 브랜드의 일부니까. 판매 전 확인해야 할 기준을 쉽게 정리합니다.

---

## 3. 디자인 원칙

- **레퍼런스**: Toss Ads Insights ([tossads.toss.im/insights](https://tossads.toss.im/insights))
- KRK Checker Guideline 컬러 사용: Heritage Blue / Breath Blue / Surface / Ink

- 각 글은 `기준 출처`를 명확히 노출 → 신뢰 형성

---

## 4. 화면 1: 블로그 리스트 (메인)

### 4.1 영역 구성 (위 → 아래)

1. **헤더 / 네비**
   - 로고: KRK Checker
   - 카테고리 탭 (가로 스크롤 가능)
   - 우측 CTA: `라벨 검토 시작하기`

2. **Featured Article 영역**
   - 최신 또는 대표 글 1개 강조
   - 큰 썸네일 + 카테고리 + 제목 + 1줄 요약 + 읽는 시간

3. **카테고리 탭 (필터)**
   - 전체 / 라벨 기준 / 원재료·알레르기 / 분리배출·포장 / 품목제조보고 / 브랜드 사례 / KRK 업데이트
   - 선택된 탭 시각적 강조

4. **Article Card 그리드**
   - Desktop: 3열 / Tablet: 2열 / Mobile: 1열
   - 카드 내 노출 요소:
     - 카테고리 라벨
     - 제목
     - 1줄 요약
     - 읽는 시간 (예: 3분)
     - 기준 출처 태그 (예: 식약처, 식품표시광고법)

5. **푸터**
   - KRK 브랜드 / 정책 / 문의

---

## 5. 화면 2: 블로그 상세 (Article Detail)

### 5.1 영역 구성 (위 → 아래)

1. **헤더 / 네비** (리스트와 동일)

2. **Article Header**
   - 카테고리 라벨
   - 제목 (대형)
   - 1줄 요약 (서브타이틀)
   - 메타: 작성일 / 읽는 시간 / 기준 출처 태그

3. **Hero 영역** (선택적)
   - 대표 이미지 또는 일러스트 영역

4. **본문 영역**
   - 마크다운 기반 콘텐츠 렌더링
   - 지원 요소: H2 / H3 / 본문 / 리스트 / 인용구 / 이미지 / 표 / 콜아웃 박스 (예: "⚠️ 놓치기 쉬운 항목")
   - 최대 폭 제한 → 가독성 우선
   - 우측 (Desktop): 목차 sticky 영역 (선택적)

5. **글 하단 CTA 블록** ⭐ 핵심
   - 메시지: "이 글의 기준대로 라벨을 검토하고 싶다면"
   - 버튼: `KRK Checker 시작하기` (Primary)
   - 또는: `라벨 검토 시작하기`
   - 시각적으로 본문과 명확히 구분되는 박스/배경

6. **관련 글 추천**
   - 같은 카테고리 또는 연관 주제 글 2~3개

7. **푸터**

## 6. 콘텐츠 카드 / 글 메타 데이터 구조

```

  카테고리: "라벨 기준",
  제목: "식품 라벨에 꼭 들어가야 하는 9가지 표시사항",
  요약: "...",
  읽는 시간: "5분",
  기준 출처: ["식품표시광고법", "식약처 고시"],
  발행일: "2026-05-24",
  썸네일: "..."
}
```

## 7. 콘텐츠 / 카피 가이드

### ✅ 권장 표현
판매 전 확인 / 표시 기준 / 자율 점검 / 놓치기 쉬운 항목 / 기준 출처 / 검토 결과는 참고용

### ❌ 금지 표현
법적 문제 해결 / 통과 보장 / 인증 완료 / 공식 검토 / 법률 자문

---

## 8. 반응형 요구사항

| Breakpoint | 그리드 | 네비 |
|---|---|---|
| Desktop (≥1024px) | 3열 카드 그리드 | 가로 카테고리 탭 |
| Tablet (768~1023px) | 2열 카드 그리드 | 가로 카테고리 탭 |
| Mobile (<768px) | 1열 카드 리스트 | 가로 스크롤 카테고리 탭 / 하단 sticky CTA 고려 |

---

## 9. 기술 / 구현 가이드

- 정적 사이트 또는 SSR 권장 (SEO 우선)
- 콘텐츠: 마크다운 기반 (MDX 가능)
- 폰트 / 컬러 / 컴포넌트: KRK Checker Guideline 따름
- 접근성: WCAG AA 수준

---

## 10. 초기 콘텐츠 10개 (참고용 — 카드 더미 데이터로 활용 가능)

1. 식품 라벨에 꼭 들어가야 하는 9가지 표시사항
2. 원재료명은 왜 함량순으로 써야 할까?
3. 알레르기 유발물질 22품목, 라벨에서 놓치기 쉬운 부분
4. 무가당이라고 쓰면 영양성분표가 필요할 수 있어요
5. 분리배출 마크, 작은 브랜드도 표시해야 할까?
6. 유리병+플라스틱 뚜껑, 포장재가 2개일 때 표시 기준
7. 원산지 표시는 언제 굵게 써야 할까?
8. 식품제조가공업과 즉판가공업, 품목제조보고 차이
9. 라벨 PDF를 디자이너에게 넘기기 전 확인할 것
10. KRK Checker는 검토 결과를 어떻게 보여주나요?
