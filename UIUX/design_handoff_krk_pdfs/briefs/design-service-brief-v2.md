## 🗺 확정된 서비스 플로우 (4단계)

```
[STEP 1~4] Creator 입력
  제품 정보 → 원재료 → 영양성분 → 라벨 미리보기
  CTA: "검토 결과 보기" → /review 이동

      ↓

[/review] 검토 결과 + 서비스 선택
  - 16개 법규 신호등 자동 결과 표시
  - 위반 항목 강조 (빨간 배너 + 과태료)
  - 서비스 선택 카드 2개: 기본(9,900) / 전문(19,900)
  - CTA: "결제하기" → /payment
  - CTA: "← 입력 수정하기" → Creator 복귀

      ↓

[/payment] 결제 화면
  - 선택한 서비스 기준 금액
  - Toss Payments 위젯

      ↓

[/payment/complete] 결과 + 다운로드 허브
  - 기본/전문에 따라 다른 파일 제공
  - 마이페이지 1년 재다운로드 안내 (현재 준비 중)
```

## 🔍 검토 항목 (16개, R01~R20)

결과 화면에서 각 항목을 신호등으로 표시:

| 상태 | 의미 | 색 |
|------|------|---|
| 🔴 위반 | 법규 위반, 즉시 수정 필요 | risk (#B30000) |
| 🟡 경고 | 확인 권장, 상황에 따라 다름 | warn (#8A5A00) |
| 🟢 통과 | 기준 충족 | Heritage (#002D72) |

**기본 결제 시**: 신호등 + 감지 이유 1줄만 표시. 수정방법·과태료·법규 조항은 잠금.
**전문 결제 시**: 모든 내용 잠금 해제.

## 🎨 화면별 핵심 카피 방향

### /review 검토 결과 화면
- 메인 메시지: 위반 발견 시 긴장감 → 결제 동기 형성
- 예시: `"판매 전 확인이 필요한 항목이 발견됐어요."` (위반 있을 때)
- 예시: `"입력하신 라벨이 기준에 맞는지 정리했어요."` (통과 시)
- 서비스 선택 카드: 각 서비스에 **받을 것 명시** → 오해 방지

### /payment/complete 다운로드 허브
- 감정선: 결제 직후 → 안도 + 신뢰
- 예시: `"전문 수정 가이드가 준비됐어요."`
- 마이페이지 안내: `"마이페이지에서 1년간 재다운로드 가능합니다."` (현재 준비 중 표기)
